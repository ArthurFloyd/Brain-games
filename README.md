## Brain-games

#### Description
*Five logic games with the ability to change the difficulty.*

#### Minimum system requirements
*__Operating system:__ macOC Ventura 13.2* 

*__Node.js:__ 20.3.1*


### Brain-even
**_start:_**
```
brain-even
```
**_rules game:_**
*Answer "yes" if the number is even, otherwise answer "no".*
